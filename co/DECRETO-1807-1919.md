# Por el cual se reorganizan unas oficinas y se crea un empleo en el ramo telegráfico

**El Presidente de la República de Colombia,**

en uso de sus facultades legales,

**decreta:**

##### **Artículo 1º** A partir del primero de octubre próximo reorganízanse las Oficinas Telegráficas de Filandia y Salento, cada una con el personal y asignaciones mensuales siguientes:

| **Filandia** |  |  |
| --- | --- | --- |
| Un Telegrafista Administrador de Correos, con | $ | 60 |
| Un Ayudante |  | 55 |
| Un Cartero |  | 6 |
| Para gasto de alumbrado |  | 3 |
| **Salento** |  |  |
| Un Telegrafista Administrador de Correos, con |  | 60 |
| Un Ayudante |  | 55 |
| Un Cartero |  | 6 |
| Para gastos de alumbrado |  | 1 |

##### **Artículo 2º** Créase una plaza más de Ayudante en la Oficina Telegráfica de Pereira, con la misma asignación mensual de $ 52, que hoy tienen los demás Ayudantes de aquella oficina.

**Comuníquese y publíquese.**

**Dado en Bogotá a 22 de septiembre de 1919.**

**MARCO FIDEL SUAREZ-El Ministro de Gobierno, Marcelino ARANGO.**

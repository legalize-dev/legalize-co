# Por el cual se distribuye la partida para sueldos y gastos varios de las Escuelas Ricaurte y Bavaria, anexas al Instituto Pedagógico Nacional

***El Presidente de la República de Colombia,***

en uso de sus facultades legales,

**DECRETA:**

##### **Articulo único.** Para sueldos del personal y gastos varios de las Escuelas "Ricaurte" y "Bavaria", anexas al Instituto Pedagógico Nacional, se distribuye la siguiente partida apropiada de Presupuesto Nacional vigente, así:

### Capítulo 63, artículo 875.

- a) Para sueldos en once (11) meses, de enero a noviembre, asi:

|  | Mensual | Total |
| --- | --- | --- |
| Dos Directores, a $ 110 cada uno | 220.00 | 2.420.00 |
| Seis Profesores de Grupo, a $ 90 cada uno | 510.00 | 5.910.00 |
| Profesor de Educación Artística | 25.00 | 275.00 |
| Portero | 30.00 | 330 00 |
| Sumas | 8 815.00 | 8.965.00 |
| b) Servicios: |  |  |
| De agua, aseo, teléfono, material de enseñanza, útiles de escritorio, deportes, excursiones y demás gastos de estas Escuelas |  | $ 875 00 |
| Total |  | $ 9 840.00 |

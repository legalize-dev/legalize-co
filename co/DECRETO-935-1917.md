# Por el cual se reorganizan dos oficinas en el ramo Telegráfico

**El Presidente de la República de Colombia,**

en uso de sus facultades legales,

**decreta:**

A partir del primero de junio próximo reorganizase las Oficinas Telegráficas de Bolívar (Cauca) y San Pablo (Nariño), cada una con el personal y asignaciones mensuales siguientes:

| **Bolívar** |  |  |  |
| --- | --- | --- | --- |
| Un Telegrafista Administrador de Correos, con | $ | 50 |  |
| Un Ayudante |  | 45 |  |
| Un Cartero |  | 3 |  |
| Para gastos de alumbrado |  | 2 | 50 |
| **San Pablo** |  |  |  |
| Un Telegrafista Administrador de Correos, con |  | 30 |  |
| Un Cartero |  | 3 |  |
| Para gastos de Alumbrado |  | 1 |  |

**Comuníquese y publíquese.**

**Dado en Bogotá a 19 de mayo de 1917.**

**JOSE VICENTE CONCHA--Por El Ministro de Gobierno, El Secretario, Juan de la Cruz DUARTE.**

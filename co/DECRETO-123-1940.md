# Por el cual se destinan las partidas para sueldos del personal del Liceo Celedón de Santa Marta

**El Presidente de la República de Colombia,**

en uso de sus facultades legales,

**decreta:**

##### **Artículo único.** Para sueldos del personal del Liceo Celedón, de Santa Marta, se destinan las siguientes sumas apropiadas en el Presupuesto vigente:

- a) Del artículo 966, Capítulo 61.

Para sueldos en doce (12) meses, de enero a diciembre, asi:

|  |  | Mensual | Total |
| --- | --- | --- | --- |
| Rector (interno), a | $ | 300.00 | 3.600.00 |
| Vicerrector (interno), a |  | 200.00 | 2.400.00 |
| Secretario Habilitado (interno), a |  | 150.00 | 1.800.00 |
| Directores de Grupo (internos), once (11), a $ 130.00 cada uno |  | 1.430.00 | 17.160.00 |
| Médico Profesor (externo), a |  | 100.00 | 1.200.00 |
| Dentista Profesor (externo), a |  | 70.00 | 840.00 |
| Bibliotecario (interno), a |  | 40.00 | 480.00 |
| Ecónomo (interno), a |  | 80.00 | 960.00 |
| Portero (interno), a |  | 50.00 | 600.00 |
| Totales | $ | 2.420.00 | 29.040.00 |

**Parágrafo.** La partida de cuarenta y un mil pesos ($ 41.000.00), que corresponde aportar al Departamento de Magdalena, conforme a las cláusulas 4ª y 10 del contrato de nacionalización, suscrito el 11 de febrero de 1938, publicado en el **Diario Oficial** número 23742, será distribuida por Resolución del Ministerio de Educación Nacional.

**Comuníquese y publíquese.**

**Dado en Bogotá, a 30 de enero de 1940.**

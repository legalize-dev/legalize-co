# Por el cual se distribuye la partida asignada en el capítulo 13, articulo 88, del Presupuesto Nacional para la presente vigencia, destinada para obras de fomento en las Intendencias y Comisarias

***El Presidente de la República de Colombia,***

en uso de sus facultades légales,

**DECRETA:**

##### **Artículo único.** Distribúyese la suma de veintitrés mil pesos (S 23.000) moneda legal, apropiada en el capítulo 13, artículo 88, del Presupuesto Nacional para la vigencia fiscal del presente año, destinada para obras de fomento en las Intendencias y Comisarías, especialmente provisión de aguas, en la siguiente forma:

| Comisaría del Caquetá | $ | 3.000.00 |
| --- | --- | --- |
| Comisaría del Vaupés |  | 3.000.00 |
| Comisaría del Vichada |  | 3.000.00 |
| Comisaría de Arauca |  | 2.000.00 |
| Comisaría del Putumayo |  | 2.000.00 |
| Comisaría de La Guajíra |  | 2.000.00 |
| Intendencia de San Andrés y Providencia |  | 2.000.00 |
| Intendencia del Amazonas |  | 2.000.00 |
| Intendencia del Chocó |  | 2.000. 00 |
| Intendencia del Meta |  | 2.000.00 |
| Total | $ | 23.000.00 |

Comuníquese y publíquese.

**Dado en Bogotá a 7 de febrero de 1941.**

**EDUARDO SANTOS**

**El Ministro de Gobierno,**

**Jorge GARTNER**

# Por el cual se suprimen unos cargos, se crean otros y se fijan unas asignaciones en la Campaña Antituberculosa Nacional

**El Presidente de la República de Colombia,**

en uso de sus atribuciones legales,

**DECRETA:**

##### **Artículo primero.** A partir del 16 de agosto del presente año, suprímense los siguientes cargos:

| *Hospital Sanatorio Santa Clara, de Bogotá.* |  |
| --- | --- |
| 1 Médico Subdirector | $ 650.00 |
| 1 Broncoscopista Auxiliar | 350.00 |
| *Hospital Sanatorio Santa Sofía, de Manizales.* |  |
| 1 Capellán | $ 200.00 |
| 1 Broncoscopista Auxiliar | 200.00 |
| *Hospital Sanatorio La María de Medellín* |  |
| 2 Médicos Internos, cada uno $200.00 $ 400.00 | $ 200.00 |
| *Hospital Sanatorio de San Pablo, de Cartagena.* |  |
| 1 Ecónoma | $ 200.00 |
| 1 Enfermera | 200.00 |
| 1 Encargada de Estadística | 200.00 |

***<<Artículo segundo.*** NO DIGITADO EN EL DIARIO OFICIAL 28859.>>

##### ***Artículo tercero.*** A partir de 16 de agosto del presente año, creánse los siguientes cargos:

| *Unidad Móvil de Antioquia.* |  |
| --- | --- |
| 1 Médico Jefe | $ 1.000.00 |
| *Hospital Sanatorio Santa Clara, de Bogotá.* |  |
| 1 Jefe del Departamento de Quirúrgico, | $ 600.00 |

##### ***Artículo cuarto.*** El gasto que ocasione el cumplimiento del presente Decreto, se liará con, cargo al Capítulo 72, artículo 1007 del Presupuesto de Gastos, vidente y a los correspondientes a las próximas vigencias.

**Comuníquese, y cúmplase.**

**Dado en Bogotá a 31 de agosto de 1955.**

**Gral. Jefe Supremo GUSTAVO ROJAS PINILLA,**

# Por el cual se distribuye una partida

**La Junta Militar de Gobierno de la República de Colombia,**

**DECRETA:**

##### ***Artículo primero.*** De la partida señalada en el Capítulo 348, artículo 3669 del Presupuesto de la actual vigencia, destinase la suma de siete mil setenta y un pesos (S 7.071.00) moneda corriente, para establecer y organizar los siguientes Cursos de Capacitación Pedagógica y mejoramiento profesional para maestros de escuela primaria en ejercicio, que estén siguiendo los Cursos Radiales, del 15 de agosto al 15 de septiembre para el Departamento del Valle, y del 15 de noviembre al 15 de diciembre para los Departamentos de Antioquia, Norte de Santander y Santander, de acuerdo con la siguiente distribución:

| **Curso de Cali:** | **Mensual:** | **Total:** |
| --- | --- | --- |
| Para sueldo de un Director | $ 700.00 |  |
| Para transporte del Director | 93.60 |  |
| Para sueldo de dos profesores de tiempo completo, a $ 500.00 cada uno | 1.000.00 |  |
| Para útiles de escritorio | 100.00 | $1.893.60 |
| **Cursos de Medellín:** | **Mensual:** | **Total:** |
| Para sueldo de cuatro profesores de tiempo completo, a $ 500.00 cada uno | 2.000.00 |  |
| Para útiles de escritorio | 200.00 | $ 2.200.00 |
| **Curso de Pamplona:** | **Mensual:** | **Total:** |
| Para sueldo de dos profesores de tiempo completo, a $ 500.00 cada uno | 1.000.00 |  |
| Para útiles dé escritorio | 100.00 | 1.100.00 |
| **Curso de Bucaramanga:** | **Mensual:** | **Total:** |
| Para sueldo de un Director | 700.00 |  |
| Para sueldo de dos profesores de tiempo completo, a $ 500.00 cada uno | 1.000.00 |  |
| Para transporte del Director | 77.40 |  |
| Para útiles dé escritorio | 100.00 | $ 1.877.40 |
| Total |  | $ 7.071.00 |

##### ***Artículo segundo.*** Las funciones de Habilitado Pagador de los Cursos quedan adscritas así:

Para el Curso de Cali, la habilitada de la Normal de Señoritas, de Cali.

Para el Curso de Pamplona, el Habilitado de la Normal Superior de Varones, de Pamplona.

Para los Cursos de Medellín, el Habilitado de la Normal de Varones, de Medellín.

Para el Curso de Bucaramanga, la Habilitada de la Normal de Señoritas de Bucaramanga.

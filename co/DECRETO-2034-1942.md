# Sobre tarifas de encomiendas postales para algunos países

***El Presidente de la República de Colombia,***

en uso de sus facultades legales,

***decreta:***

##### **Artículo 1º** A partir del lº de septiembre del año en curso regirán las siguientes tarifas para las encomiendas postales que de Colombia se despachen a los siguientes países:

| *Argentina*: |  |  |
| --- | --- | --- |
| Encomiendas para localidades de la Costa del Sur, Tierra del Fuego e islas adyacentes: |  |  |
| Encomiendas hasta 1 kilogramo de peso | $ | 5 05 |
| Encomiendas de más de 1 kilogramo hasta 3 kilogramos de peso |  | 5 65 |
| Encomiendas de más de 3 kilogramos hasta 5 kilogramos de peso |  | 6 12 |
| Para otras localidades: |  |  |
| Encomiendas hasta 1 kilogramo de peso |  | 4 60 |
| Encomiendas de más de 1 kilogramo hasta 3 kilogramos de peso |  | 5 20 |
| Encomiendas de más de 3 kilogramos hasta 5 kilogramos de peso |  | 5 65 |
| *Bolivia:* |  |  |
| Para La Paz y Oruro: |  |  |
| Encomiendas hasta 1 kilogramo de peso |  | 1 94 |
| Encomiendas de más de 1 kilogramo hasta 3 kilogramos de peso |  | 5 20 |
| Encomiendas de más de 3 kilogramos hasta 5 kilogramos de peso |  | 2 50 |

| *Uruguay* |  |  |
| Encomiendas hasta 1 kilogramo de peso |  | 5 05 |
| Encomiendas de más de 1 kilogramo hasta 3 kilogramos de peso |  | 5 65 |
| Encomiendas de más de 3 kilogramos hasta 5 kilogramos de peso |  | 6 10 |

##### **Artículo 2º** Quedan así derogadas las tarifas para las encomiendas postales destinadas a la Argentina, Bolivia, Chile. Paraguay. Perú y Uruguay, señaladas en el Decreto número 607 de 6 de marzo del año en curso.

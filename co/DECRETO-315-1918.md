# Por el cual se reorganizan unas oficinas en el ramo Telegráfico

**El Presidente de la República de Colombia**

en uso de sus facultades legales,

**decreta:**

Reorganízanse las Oficinas Telegráficas de Garagoa, Guateque y Chinavita, cada una con el personal y asignaciones mensuales siguientes:

| Garagoa | Garagoa | Garagoa |
| --- | --- | --- |
| Un Telegrafista Jefe, con | $ | 30 |
| Un Ayudante |  | 20 |
| Un Cartero |  | 3 |
| Guateque. | Guateque. | Guateque. |
| Un Telegrafista Administrador de Correos, con |  | 45 |
| Un Ayudante |  | 35 |
| Un Cartero |  | 4 |
| Chinavita. | Chinavita. | Chinavita. |
| Un Telegrafista Administrador de Correos, con |  | 45 |
| Un Ayudante |  | 35 |
| Un Cartero |  | 3 |

**Comuníquese y publíquese.**

**Dado en Bogotá a 20 de febrero de 1918.**

**JOSE VICENTE CONCHA-El Ministro De Gobierno, Miguel ABADIA MÉNDEZ.**

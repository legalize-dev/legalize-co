# Por el cual se suprimen y crean unos cargos de la Planta de Personal de la Dirección General del Presupuesto del Ministerio de Hacienda y Crédito Público

**El Presidente de la República de Colombia,**

en uso de las facultades que le confiere el numeral 21 del artículo 120 de la Constitución Política de Colombia y de acuerdo con los Decretos leyes 1912 de 1973 y 540 de 1977,

| Nº de cargos, | Denominación del empleo | Clase | Grado | Sueldo Básico |
| --- | --- | --- | --- | --- |
|  | **Dirección General del Presupuesto** |  |  |  |
|  | Subdirección de Ejecución Presupuestal |  |  |  |
|  | Divisiones Delegadas de Presupuesto |  |  |  |

| 1 | Técnico en Contabilidad | II | 14 | 6.500 |

| Nº de cargos, | Denominación del empleo | Clase | Grado | Sueldo Básico |
| --- | --- | --- | --- | --- |
|  | **Dirección General del Presupuesto** |  |  |  |
|  | División Administrativa |  |  |  |
| 3 | Programador de Equipo de Sistematización | III | 16 | 7.800 |

##### **Artículo 3º** El presente Decreto rige a partir de la fecha de su expedición.

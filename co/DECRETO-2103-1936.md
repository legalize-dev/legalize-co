# Por medio del cual se distribuye una partida

**El Presidente de la República de Colombia,**

en uso de sus atribuciones legales,

**DECRETA:**

##### **Artículo único.** De acuerdo con el artículo 27 del Decreto número 1528 de 25 de junio del presente año, destínanse $ 50,000 que se tomarán del capítulo 52, artículo 318, del Presupuesto de la actual vigencia, para el Instituto Nacional de Educación Física y que serán distribuidos en la siguiente forma:

| Para personal administrativo | $ | 2.125 |
| --- | --- | --- |
| Para el profesorado |  | 3.500 |
| Para dotaciones generales |  | 6.000 |
| Para Reformas en el local que ocupara en la Facultad de Medicina |  | 6.000 |
| Para gastos varios e imprevistos |  | 2.375 |
| Para la construcción del edificio en la Ciudad Universitaria |  | 30.000 |
|  | $ | 50.000 |

**Comuníquese y publíquese.**

**Dado en Bogotá a 26 de agosto de 1936.**

**ALFONSO LOPEZ**

**Por el Ministro de Educación Nacional, el Secretario, encargado,**

**Jorge ZALAMEA**

# por el cual se reorganiza la Oficina de Correos de Manizales

***El Presidente de la República de Colombia,***

en uso de sus facultades legales,

**DECRETA:**

##### **Artículo 1.°** A contar del 1.° de julio próximo se reorganiza la Administración Principal de Correos Nacionales de Manizales, con el personal y asignaciones mensuales que se expresan:

| Un Administrador Principal, con | $ | 80 |
| --- | --- | --- |
| Un Tenedor de Libros, con |  | 50 |
| Un Auxiliar de éste, con |  | 30 |
| Un Oficial de Correspondencia, con |  | 35 |
| Un Oficial de Encomiendas, con |  | 45 |
| Un Oficial de Recibo y Despacho, con |  | 35 |
| Un encargado de los Giros Postales y Expendedor de Especies, con |  | 25 |
| Un Portero Escribiente, con |  | 20 |
| Para útiles de escritorio |  | 15 |

##### **Artículo 2.°** Por la Dirección General de Correos y Telégrafos se harán los nombramientos a que da lugar el presente Decreto.

**Comuníquese y publíquese.**

**Dado en Bogotá a 22 de junio de 1912.**

**CARLOS E. RESTREPO**

**El Ministro de Gobierno,**

**pedro m. CARREÑO**

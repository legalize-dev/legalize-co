# Por el cual se aumenta el precio de la sal

***El Vicepresidente de la República, encargada del Poder Ejecutivo,***

**CONSIDERANDO:**

1.° Que según informe del Sr. Administrador principal de las Salinas de Cundinamarca es preciso y envíale, aumentar el precio que actualmente tiene la sal que el Gobierno vende, y

2.° Que subsisten y aun han aumentado las causas que motivaron los Decretos expedidos anteriormente en este mismo asunto,

**DECRETA:**

Desde la fecha del presente Decreto los precios de la sal, en sus distintas formas, serán los siguientes:

| Cada 12½ kilogramos de sal compactada | $ | 10 |
| --- | --- | --- |
| Cada 12½ kilogramos de sal de grano de caldero |  | 7 |
| Cada id. id. vijua de 1.a |  | 4 |
| Cada id. id. vijúa de 2.a |  | 3 |
| Cada decálitro de agua salada, á 16° Baume |  | 0 70 |
| Cada id. id. á20° id |  | 0 80 |
| Cada id. id. á 25° íd |  | 105 |

**Comuníquese y publíquese.**

**Dado en Bogotá, á 18 de Octubre de 1902.**

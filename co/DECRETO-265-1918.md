# Por el cual se reorganiza una oficina en el ramo Telegráfico

**El Presidente de la República de Colombia,**

en uso de sus facultades legales,

**decreta:**

A partir del primero de marzo próximo, reorganízase la Oficina Telegráfica de Santa Marta, con el personal y asignaciones mensuales siguientes:

| Un Jefe, con | $ | 65 |
| --- | --- | --- |
| Dos Ayudantes, cada uno con $ 55. |  | 110 |
| Un Oficial de Recibo |  | 52 |
| Un Copista |  | 40 |
| Dos Carteros, cada uno a $ 14. |  | 28 |

**Comuníquese y publíquese.**

**Dado en Bogotá a 14 de febrero de 1918.**

**JOSE VICENTE CONCHA-El Ministro de Gobierno, miguel ABADIA MENDEZ.**

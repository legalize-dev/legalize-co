# Por el cual se reorganiza la Administración de Correos de Cúcuta

***El Presidente de la República de Colombia,***

en uso de sus facultades legales,

**DECRETA:**

##### **Artículo 1.°** Reorganízase la Administración Principal de Correos de Cúcuta con el siguiente personal y asignaciones mensuales:

| Un Administrador Principal con | $ | 105 |
| --- | --- | --- |
| Un Contador Tenedor de Libros, con |  | 70 |
| Un Auxiliar de éste, con |  | 36 |
| Un Oficial de Correspondencia y Encomiendas, con |  | 45 |
| Un Oficial de Recomendados y Valores, con |  | 45 |
| Un Portero Escribiente encargado de la Sección de Lista, con |  | 25 |
| Un Jefe del Correo Urbano, con |  | 40 |
| Cuatro Carteros y Buzoneros, a $12 cada uno |  | 48 |
| Para útiles de escritorio |  | 20 |

##### **Artículo 2.°** La Dirección General de Correos y Telégrafos hará las designaciones a que da lugar el presente Decreto.

**Comuníquese y publíquese.**

**Dado en Bogotá a 21 de junio de 1912.**

**CARLOS E. RESTREPO**

**El Ministro de Gobierno,**

**pedro m. CARREÑO**

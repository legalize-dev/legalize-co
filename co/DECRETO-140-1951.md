# Por el cual se distribuyen unas apropiaciones (Ramo Carreteras Nacionales)

**El Presidente de la República de Colombia,**

en uso de sus atribuciones legales,

**DECRETA**:

##### **Artículo único.** De la partida apropiada en el Capítulo 118, artículo 1581, del Presupuesto de la vigencia fiscal de 1951, "para la terminación de puentes y construcción de otros", háncese la siguiente distribución parcial para la ejecución de las siguientes obras:

|  | *Capítulo 118-Articulo 1581.* |  |  |
| --- | --- | --- | --- |
| Ordinal a) | Para la terminación del puente de el Comercio, sobre el rio Cauca | $ | 324.000.00 |
| Ordinal b) | Para el puente de Honda |  | 450.000.00 |
| Ordinal c) | Para el puente río Neiva |  | 91.000.00 |
|  |  |  | 865.000.00 |

**Comuníquese y publíquese.**

**Dado en Bogotá a 23 de enero de 1951.**

**LAUREANO GOMEZ**

**El Ministro de Hacienda y Crédito Público, Rafael DELGADO BARRENECHE-El Ministro de Obras Públicas, Jorge LEYVA.**

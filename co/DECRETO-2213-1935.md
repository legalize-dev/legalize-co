# Por el cual se determinan las secciones que forman el Departamento Administrativo de la Policía Nacional y se fija su personal y asignaciones

***El Presidente de la República de Colombia,***

en uso de las autorizaciones que le confiere la Ley 15 del presente año,

**DECRETA:**

##### **Artículo 1º.** Desde el día 16 del presente mes, el Departamento Administrativo de la Policía Nacional, creado por el Decreto número1994 de 1935, tendrá además de la Jefatura, las siguientes Secciones, personal y asignaciones mensuales:

| SECCION 1ª. |  |
| --- | --- |
| *Contabilidad y Control.* |  |
| Un Contador Jefe | $ 225 |
| Un Inspector General | 150 |
| Cuatro Contadores Auxiliares, cada uno | 150 |
| Un Oficial de cuentas de caja | 120 |
| Un Oficial de cuentas de Intendencia | 120 |
| Un Oficial de cuentas de Proveeduría | 120 |
| Un Oficial de cuentas de Casinos | 120 |
| Un Oficial de reparto y archivo | 80 |
| Un Mecanógrafo | 70 |

| Un Cajero General | 250 |

| Tres Ayudantes Pagadores, cada uno a | 100 |
| Un Oficial de reparto y archivo | 80 |
| Un Mecanógrafo | 70 |
| SECCION 3ª |  |
| *Intendencia.* |  |
| Un Intendente | 250 |
| Un Secretario | 20 |
| Dos Almacenistas de material y equipo, cada uno | 100 |
| Un Almacenista de depósitos judiciales | 100 |
| Un Sastre | 80 |

| Un Maestro Carpintero | 70 |

| Un Conserje | 35 |
| Un Sirviente | 15 |
| SECCION 4ª: |  |
| *Proveeduría.* |  |
| Un Proveedor | 250 |
| Un Secretario | 120 |
| Un Liquidador | 100 |
| Dos Almacenista, cada uno a | 80 |
| Dos Pesadores, cada uno a | 55 |
| Un Portero | 55 |
| Un Sirviente | 15 |

##### **Artículo 2°.** Desde la fecha citada en el artículo anterior, quedan suprimidos las Secciones de Contabilidad, Control y Estadística, Habitación, Proveeduría y Casinos que funcionan actualmente en la Policía Nacional.

##### **Artículo 3°.** Desde la vigencia de este Decreto queda derogado el artículo 13 del Decreto 837 de 26 de mayo de 1930.

**Comuníquese y publíquese.**

**Dado en Bogotá, a 11de diciembre de 1935.**

**ALFONSO LOPEZ**

# por el cual se crean unas Oficinas en el ramo Telegráfico

**El Presidente de la República de Colombia,**

en uso de sus facultades legales,

**decreta:**

Créanse las Oficinas Postales y Telegráficas de Ricaurte (Caquetá) y Miranda (Departamento del Cauca), cada una con el personal y asignaciones mensuales siguientes:

| **Ricaurte.** | **Ricaurte.** | **Ricaurte.** |
| --- | --- | --- |
| Un Telegrafista Administrador de Correos, con | $ | 25 |
| Un Cartero |  | 3 |
| Para gastos de alumbrado |  | 1 |
| **Miranda.** | **Miranda.** | **Miranda.** |
| Un Telegrafista Administrador de Correos, con | $ | 22 |
| Un Cartero |  | 3 |
| Para gastos de alumbrado |  | 1 |

**Comuníquese y publíquese.**

**Dado en Bogotá a 16 de marzo de 1916.**

**JOSE VICENTE CONCHA - El Ministro de Gobierno, Miguel ABADIA MENDEZ.**

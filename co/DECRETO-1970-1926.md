# Por el cual se reorganiza la construcción del ferrocarril Ibagué- Armenia y carretera auxiliar

**El Presidente de la República de Colombia,**

en uso de sus facultades legales,

decreta:

##### **Artículo 1°** A partir del primero de diciembre próximo, redúcese el personal técnico para la construcción del ferrocarril Ibagué-Armenia y su carretera auxiliar, personal que quedará reorganizado en la forma y con las asignaciones que a continuación se expresan:

| Un Ingeniero Director, con | $ | 500 |
| --- | --- | --- |
| Un Ingeniero Jefe de trazado, con |  | 400 |
| Un Ingeniero primer Ayudante, con |  | 300 |
| Un Ingeniero segundo Ayudante, con |  | 240 |
| Dos Dibujantes, cada uno con |  | 180 |
| Un Ingeniero adjunto de construcción, con |  | 400 |
| Un Ingeniero Ayudante de construcción, con |  | 250 |
| El Ingeniero Director queda facultado para distribuir el personal técnico de acuerdo con las exigencias de los trabajos. | El Ingeniero Director queda facultado para distribuir el personal técnico de acuerdo con las exigencias de los trabajos. | El Ingeniero Director queda facultado para distribuir el personal técnico de acuerdo con las exigencias de los trabajos. |

##### **Articulo 2°** Nómbrase para desempeñar estos puestos a los siguientes señores:

Ingeniero Jefe de trazado, José Vicente Dávila.

Ingeniero primer Ayudante, Guido Sanna Solinas.

Ingeniero segundo Ayudante, Fernando Jaramillo A.

Dibujantes, Carlos E. Venegas y Hernando Salazar.

Ingeniero adjunto de construcción, Federico Pin.

Ingeniero Ayudante de construcción, Valerio Convers.

El cargo de Ingeniero Director continuará desempeñándolo el doctor César A. Pedraza.

**Comuníquese y publíquese.**

**Dado en Bogotá a 25 de noviembre de 1926.**

**MIGUEL ABADIA MENDEZ-El Ministro de Obras Públicas, Mariano OSPINA PEREZ.**

# Por el cual se reorganiza la Administración de Correos de Bucaramanga y se suprimen los tres Carteros de la Oficina Telegráfica de dicha ciudad

en uso de sus facultades legales,

**DECRETA:**

##### **Artículo 1.°** Reorganízase la Administración de Correos de Bucaramanga con el siguiente personal y asignaciones mensuales:

| Un administrador Principal, con | $ | 100 |
| --- | --- | --- |
| Un Tenedor de Libros, con |  | 54 |
| Un Auxiliar de éste, con |  | 36 |
| Un Oficial de Correspondencia, con |  | 32 |
| Un Oficial de Encomiendas, con |  | 32 |
| Un Oficial de Recibo y Despacho, con |  | 40 |
| Un Portero Escribiente, con |  | 20 |
| Un Encargado del servicio de giros postales, con |  | 20 |
| Para útiles de escritorio |  | 15 |

##### **Artículo 2.°** Establécese el Correo Urbano Nacional para la distribución de la correspondencia postal y telegráfica en la misma ciudad de Bucaramanga, y se dispone que sea servido por el siguiente personal, al cual se fijan las asignaciones mensuales que se expresan:

| Un Jefe, con | $ | 50 |
| --- | --- | --- |
| Un Subjefe Ayudante, con |  | 30 |
| Seis Carteros y dos Buzoneros, a $12 cada uno |  | 96 |
| Para útiles de escritorio |  | 2 |

La Dirección General de Correos y Telégrafos hará las designaciones a que da lugar el presente Decreto:

##### **Artículo 3.°** Se faculta igualmente a la Dirección General de Correos y Telégrafos para hacer los gastos que ocasione la instalación del Correo Urbano de que se ha hablado.

##### **Artículo 4.°** Dese la fecha en que el Correo Urbano de Bucaramanga comience a hacer el servicio de distribución de correspondencia y telegramas, se considerarán suprimidos los tres Carteros que funcionan actualmente en la Oficina Telegráfica de aquella ciudad.

**Comuníquese y publíquese.**

**Dado en Bogotá a 13 de mayo de 1912.**

**CARLOS E. RESTREPO**

**El Ministro de Gobierno,**

**pedro m. CARREÑO**

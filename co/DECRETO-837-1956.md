# Por el cual se hace una traslación en el Presupuesto de Gastos vigente (Ministerio de Minas y Petróleos)

**El Presidente, de la República de Colombia,**

en uso de sus atribuciones legales, y de conformidad con el Decreto legislativo número 1599 de 1953,

**DECRETA:**

##### **Artículo primero.** Hácese la siguiente traslación en el Presupuesto de Gastos vigente:

| MINISTERIO DE MINAS PETROLEOS |  |  |
| --- | --- | --- |
| CAPITULO 325 |  |  |
| *Laboratorio Químico Nacional.* |  |  |
| 201 Del artículo 3278. Para compra de vehículos para el servicio del Laboratorio Químico Nacional y sus dependencia, en el año | $ | 5.000.00 |
| Suma el contracrédito | $ | 5.000.00 |
| CAPITULO 320 |  |  |
| Gabinete del Ministro y Secretaria General |  |  |
| 201 Al artículo 3206. Para compra de vehículos para el servicio del Gabinete del Ministro y la Secretaria General, en el año, | $ | 5.000.0 |
| Suma el crédito | $ | 5.000.0 |

##### **Artículo segundo.** El presente Decreto rige desde la fecha de su expedición.

**Comuníquesele y publíquese.**

**Dado en Bogotá 12 de abril de 1956.**

**Gral Jefe Supremo GUSTAVO ROJAS. PINILLA,**

**Presidente de Colombia**

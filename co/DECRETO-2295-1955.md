# Por el cual se conceden una autorización a la Empresa Colombiana de Petróleos

El Presidente de la República de Colombia,

en uso de sus facultades legales, y en especial de las que le confiere el Decreto legislativo número 2894 de 1954, y

**DECRETA:**

**que** la Empresa Colombia de Petróleos ha solicitado licencia para importar accesorios varios, y tuberías de acero, con destino al campo petrolero de "El Centro", Santander del Sur, y demostró que Acerías Paz del Río, S. A. no puede venderle oportunamente tales artículos,

| **Unidades.** | **Descripción.** |  |
| --- | --- | --- |
| 600 | Pies de tubería de remoción, de junta lisa, de 8½ diámetro exterior por 7-1/8 diámetro interior. 30' de largo, grado número 80, para lavaje de collares de perforación en huecos de 8½ diámetro. |  |
| 25 | Crucelas (arm-cross), de acero galvanizado, angulares de 13" x 13" x 1-/4", 2 huecos para pasadores de 24" de distancia largo total 28" Número Seyler Mig. Co. 6622. |  |
| 55 | Crucetas (arm-cross), de acero galvanizado angular, de 3/4" x 3/4" x 1/4", dos huecos para pasadores a 48" de distancia, largo total 52". Número Seyler Mfg. Co. 6642. |  |
| 95 | Abrazaderas para crucetas (braces crossarm), de acero galvanizado plano, puntas redondas de 20", largo de 1/4" x 1/4".Número Seyler Mfg. Co. 7120. |  |
| 95 | Anclajes de expansión (anchor expansión), de acero de 7", para vástagos de 5/8" a 3/4". Número James R. Kearney número 7104. |  |
| 95 | Anclajes para varillas (rod anchor), con argolla de 5/'8" diámetro x 8' de largo. |  |

| 17 | Soportes de acero (bracket mounting), para montar 3 o 4 condensadores en cada uno, tipo F. P. Número Westinghouse 1019426, incluyendo tornillos. |  |

##### ***Artículo segundo.*** Este Decreto rige desde la fecha de su expedición.

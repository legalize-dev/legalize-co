# Por el cual se determina el personal y asignaciones de la Dirección de Información y Propaganda

**El Designado, encargado de la Presidencia de la República de Colombia,**

en uso de las facultades que le confiere el Decreto número 1102 del 29 de abril de 1952,

**DECRETA**:

##### **Articulo 1.º** El personal y asignaciones de la Dirección de Información y Propaganda de la Presidencia de la República serán las siguientes:

| --- | --- | --- |
| Un Secretario |  | 1.000.00 |
| Un Jefe de Redacción |  | 600.00 |
| Un Dibujante |  | 400.00 |
| Dos Mecanotaquígrafas, cada una |  | 330.00 |
| Un Redactor |  | 300.00 |
| Un Chófer 1.º |  | 250.00 |
| Dos Fotógrafos, cada uno |  | 200.00 |
| Dos Carteros, cada uno |  | 190.00 |

**Comuníquese y publíquese.**

**Dado en Bogotá a 5 de junio de 1952.**

**ROBERTO URDANETA ARBELAEZ**
